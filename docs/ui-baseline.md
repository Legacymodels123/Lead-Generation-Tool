# UI Baseline

## Design direction

Use an Airtable/HubSpot/Excel hybrid:
- Airtable-like property flexibility
- HubSpot-like CRM clarity
- Excel-like editing speed

## Core shell

- left navigation rail
- top workspace toolbar
- large center grid
- right contextual side panel

## UX priorities

- speed and fluidity over ceremony
- keyboard-friendly editing
- instant property add/delete/rename/reorder
- visible save state without noise
- low-friction import flow
- dynamic schema by default

## Board behavior

- rows represent companies
- columns represent dynamic properties
- property types include text, number, url, dropdown, date, boolean, ai_enriched
- AI property columns can be refreshed independently
- import can create new properties automatically

## Visual rules

- calm neutral background
- strong hierarchy around the active grid
- minimal dashboard clutter
- sticky toolbar and contextual side panel
- clear hover, selected, saving, synced, and error states

## Setup experience

Setup must remain lightweight:
- Claude
- Hunter
- Lusha
- HubSpot

Each card should explain:
- what the provider does
- which actions it unlocks
- whether it is connected
- what still needs configuration
